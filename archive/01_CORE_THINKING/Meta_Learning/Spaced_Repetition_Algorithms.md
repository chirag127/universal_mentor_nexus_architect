# Spaced Repetition Algorithms

## SUMMARY
Spaced Repetition Algorithms (SRAs) are a learning technique that leverages the psychological spacing effect to optimize memory retention. SRAs present information to learners at progressively increasing intervals, coinciding with the point where the learner is about to forget. This involves scheduling reviews of learned material based on a calculated "forgetting curve" for each item. The core principle is to revisit information just before it's forgotten, thereby strengthening the memory trace more efficiently than cramming or reviewing at fixed intervals. Common algorithms include Leitner System (card-based), SM-2 (SuperMemo 2), and Anki's proprietary implementation, which dynamically adjust review intervals based on user performance (e.g., ease of recall, number of correct answers).
